# Microprocessors and Microcontrollers

This repository contains practical **projects** developed for the *Microprocessors and Microcontrollers* course during my **Industrial Electronics and Automation Engineering degree**

The focus of this repository is embedded systems programming, hardware interaction, and low-level control using Arduino and Raspberry Pi platforms.

---

## Arduino Projects (C++)

- LED Traffic Light System
- 4x4 Keypad Access Control System
- PIR Motion Detection Alarm
- Ultrasonic Obstacle Detector
- Gas Detection System with Multi-Level Alarm
- Servo-controlled Ventilation System
- Interrupt-based Alarm Stop System

Concepts Covered:
- Digital and analog I/O
- Interrupt handling
- PWM and signal generation
- Serial communication
- Sensor integration
- Actuator control
- Embedded C++ programming

---

## Raspberry Pi Projects (Python)

- GPIO configuration and control
- Button-controlled LED systems
- Pull-up and Pull-down configurations
- Digital input monitoring
- Event-driven logic programming

Concepts Covered:
- GPIO setup and management
- Python-based hardware interaction
- Real-time signal reading
- Structured embedded scripting
